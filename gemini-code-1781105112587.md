# Project Specification: Gym Workout Assistant Web App

## 1. Project Overview
Build a responsive, modern single-page web application (SPA) to help users track and view their 3-day workout routines based on their gender. 

### Tech Stack Suggestion:
- **Frontend:** React.js or Next.js with Tailwind CSS (Alternatively, a single clean HTML/JS/Tailwind CDN file if requested for lightweight deployment).
- **Icons:** Lucide React or FontAwesome.
- **Design Style:** Dark mode preferred (Fitness/Gym vibe: Slate/Zinc background with Emerald or Amber accent colors).

---

## 2. UI/UX Workflow & Requirements

### Screen 1: Gender Selection (Landing View)
- A clean, motivational welcome screen.
- Two large interactive cards/buttons: **"NAM (MALE)"** and **"NỮ (FEMALE)"** with distinct visual cues or icons.
- Clicking a gender transitions smoothly to Screen 2.

### Screen 2: Workout Options (Dashboard View)
- Displays a top bar with a "Back" button to change gender selection and a label showing the active gender.
- A tab system or card grid presenting **3 Options (Day 1, Day 2, Day 3)** specific to that gender.
- Clicking an Option reveals the detailed list of exercises for that day.

### Screen 3: Exercise Detail Table
For each selected day, render a highly scannable grid or table containing:
- **Exercise Name** (Tên bài tập)
- **Sets x Reps** (Số hiệp x Số lần)
- **Rest Time** (Thời gian nghỉ) - *If available*
- **Notes/Target** (Ghi chú / Mục tiêu)
- **Image Placeholder:** A placeholder container for illustrative images/GIFs (`src="https://via.placeholder.com/150"` or similar layout box so the user can replace it easily later).
